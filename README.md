# 🚀 Roti Bakar Tarzz — Premium Website Template (Digital Product)

Terima kasih telah memilih **Roti Bakar Tarzz**! Template ini dirancang khusus untuk pengusaha F&B yang menginginkan kehadiran digital yang premium, modern, dan sangat fungsional.

Project ini adalah solusi *turn-key* berbasis React + Vite yang siap di-deploy untuk bisnis kuliner Anda.

---

## 🌟 Fitur Utama (USP)

- **Artisan UI/UX**: Desain mewah dengan tipografi modern dan palet warna yang menggugah selera.
- **Dynamic Cart System**: Perhitungan otomatis untuk menu utama, selai, dan topping.
- **Fully Responsive**: Pengalaman mulus di Smartphone, Tablet, maupun Desktop.
- **Ultra Fast**: Dibangun dengan Vite untuk loading speed di bawah 1 detik.
- **Easy Customization**: Semua data menu dipusatkan dalam satu file JavaScript sederhana.

## 🚀 Cara Menjalankan Project

Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/).

1. **Persiapan**:
   ```bash
   npm install
   ```

2. **Mode Pengembangan**:
   ```bash
   npm run dev
   ```
   Akses di: `http://localhost:5173`

3. **Build untuk Produksi**:
   ```bash
   npm run build
   ```
   Hasil build akan ada di folder `dist/` yang siap diunggah ke hosting (Netlify, Vercel, dll).

## 🛠️ Cara Kustomisasi (Panduan Pembeli)

### 1. Mengubah Menu, Harga, & Gambar
Anda tidak perlu menyentuh kode React yang kompleks. Cukup edit file:
`src/data/data.js`

Di sana Anda bisa mengubah:
- **Daftar Roti**: Nama, deskripsi, harga, dan ikon.
- **Pilihan Selai**: Menambah atau menghapus varian rasa.
- **Topping**: Mengatur harga tambahan untuk setiap topping.

### 2. Mengubah Nama Brand & Identitas
Buka file `src/components/` (seperti Header.jsx atau Footer.jsx) untuk mengubah teks nama brand. Untuk logo, ganti file di folder `src/assets/`.

### 3. Mengubah Warna Utama
Warna tema diatur menggunakan CSS Variables di `src/index.css`. Cari bagian `:root` untuk mengubah warna primer (gold/brown) sesuai brand Anda.
